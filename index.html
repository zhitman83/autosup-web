Главная

Дашборд вашего автомобиля

Главный экран показывает всё важное об активном автомобиле одним взглядом. Карточка авто с маркой, моделью и фотографией, текущий пробег, быстрые кнопки для добавления записей и сводная статистика по всем разделам.

  • Карточка автомобиля — марка, модель, год, госномер, фото. Быстрый доступ к редактированию.
  • Пробег — текущий одометр и кнопка обновления показаний.
  • Быстрые действия — добавить заправку, ТО или расход прямо с главного экрана.
  • Последняя заправка — дата, литры, сумма и АЗС последней заправки.
  • Сетка ТО — статусы ключевых работ: масло, шины, тормоза, ремень ГРМ и другие. Цветовая индикация: зелёный / жёлтый / красный.
  • Алерты документов — уведомления о скором истечении страховки, техосмотра, водительских прав.

Топливо

Заправки, расход и аналитика

Раздел для детального учёта заправок. Интерактивные графики по дням и месяцам, статистика расхода топлива, топ АЗС по частоте посещений.

  • График по дням — столбчатая диаграмма расхода в литрах за каждую заправку. Тап по столбцу — тултип с датой, литрами, суммой и АЗС.
  • График по месяцам — расход за 1, 6 месяцев или год. Навигация по периодам.
  • График расхода — динамика л/100км во времени, автоматический расчёт по показаниям одометра.
  • Топ АЗС — список станций с количеством заправок и средней ценой.
  • История заправок — полный список с датой, литрами, типом топлива, ценой и пробегом. Свайп для удаления.
  • Добавление заправки — дата, литры, сумма, тип топлива, пробег, АЗС. Автодополнение названий станций.

Сервис

Техническое обслуживание

Полная история обслуживания автомобиля. Каждый вид работ контролируется по пробегу и дате с цветовой индикацией статуса.

  • Список работ ТО — замена масла, фильтров, тормозных колодок, свечей, ремня ГРМ, шин и другие виды обслуживания.
  • Статус каждой работы — зелёный (в норме), жёлтый (скоро нужно), красный (просрочено) на основе интервала по пробегу или дате.
  • Интервалы — настройка по километрам и/или месяцам для каждого вида работ.
  • Мастерская — фиксирует где выполнялась работа.
  • Стоимость — сумма каждой работы для подсчёта расходов на обслуживание.
  • Добавление записи — дата, вид работы, пробег, мастерская, стоимость, примечание.

Виды работ

  • Замена масла и фильтров
  • Тормозные колодки и диски
  • Свечи зажигания
  • Ремень / цепь ГРМ
  • Шины (сезонная смена)
  • Аккумулятор
  • Охлаждающая жидкость
  • Любые другие работы

Журнал

Все события и аналитика

Сводный раздел с полной историей автомобиля. Три вкладки — Топливо, Сервис, Расходы — с месячной навигацией, сводной статистикой и экспортом отчётов.

  • Сводная карточка — donut-диаграмма расходов за месяц: заправки / ТО / доп. расходы. Тап по секции переключает вкладку.
  • Навигация по месяцам — стрелки для переключения периода, данные пересчитываются мгновенно.
  • Фильтр по категориям — во вкладке «Расходы» тап по категории фильтрует список.
  • Тренды ▲▼ — сравнение с предыдущим месяцем для каждой категории расходов.
  • Экспорт отчётов — PDF и CSV за любой период. Топливо, ТО, расходы или сводный отчёт.

Вкладки журнала

  • Топливо — сумма ₸, литры, средний расход за месяц
  • Сервис — сумма ТО, количество работ
  • Расходы — разбивка по категориям с горизонтальными барами и %

Форматы экспорта

  • PDF — красиво оформленный отчёт с шапкой авто и итогами
  • CSV — табличные данные для Excel или Google Sheets
  • Сравнение авто — сводный отчёт по всем автомобилям

Профиль

Авто, аккаунт и настройки

Управление автомобилями, аккаунтом Firebase и настройками приложения. Быстрое переключение между несколькими машинами.

  • Мои автомобили — список всех добавленных авто с активным выделенным. Переключение одним тапом, редактирование и удаление.
  • Аккаунт — вход через Google, ручная синхронизация данных с Firebase, выход из аккаунта.
  • Настройки — тема (тёмная/светлая/системная), стиль (Classic/Glass/Glass Iris), язык, единицы измерения, уведомления.
  • Сайт приложения — ссылка на эту страницу.
  • Написать нам — связь с поддержкой по email.
  • Конфиденциальность и условия — правовые документы.

Умные уведомления

Экспорт PDF и CSV

Выгружайте отчёты по заправкам, ТО, расходам или сводный отчёт. Гибкий выбор периода: месяц, 3 месяца, год или произвольный.

Категории расходов

Страховка, штрафы, мойка, парковка, запчасти — любые расходы по категориям. Переименование и объединение категорий.

Аналитика и графики

Интерактивные графики расхода топлива по дням и месяцам, динамика л/100км, donut-диаграмма расходов с трендами.

Документы с контролем сроков

Страховка, техосмотр, права и другие документы. Приложение предупреждает за 30 дней до истечения срока.
